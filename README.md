# _Pizza-Order_

#### 09-29-17

#### By _**Sravanthi Velaga**_

## Description

_A simple website that lets the user select the type, size, toppings and the number of pizzas based on which the cost of the pizza is calculated and displayed to the user._

## Specs

* _Capturing the input from the user to the Javascript_
* _Based on the size, toppings cheese and number of pizzas ordered, calculate the cost of pizza_
* _Display the size, number of pizzas ordered and the cost to the user_
